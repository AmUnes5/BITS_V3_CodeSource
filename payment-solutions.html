Payment Solutions

EPS is based on a full web J2EE Service Oriented Architecture that provides both efficiency and business flexibility :

  • Compatible with any database management system.
  • Compatible with any operating system for processing servers.
  • Compatible with any application server.

Issuing

Enables any financial and commercial institution to instantly issue, personalize and authorize any type of payment instrument compliant with any channel, any country, any currency and any network.

Securing

Helps building trust in online payments with the latest cyber security technologies of the market, namely strong authentication, one-time passwords, tokenization, data encryption, anti-fraud and anti-money laundering algorithms.

Acquiring

Supports real time acquisition of all types of transactions through any channel. It helps merchant acquirers to reduce operational costs namely for editing contracts, managing fees and remotely configuring POS.

Processing

Supports all back-office processing features, including routing domestic and international transactions, managing compensation and resolving disputes between all actors of the payment ecosystem.
